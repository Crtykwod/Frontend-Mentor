Frontend Mentor - Challenges

Oi, tudo bem?

Here, you wil find all my progress through Frontend Mentor Challenges:

GitHub Repo: https://github.com/Crtykwod/Frontend-Mentor

#1 - Newbie

  • QR Code Challenge

    This challenge consisted in creating a simple card and place a QR Code image inside it. It was an initial challenge that helped me understading better some display: flex; concepts.

    Useful Links:

    • QR Code - Frontend Mentor Solution Page
    • QR Code - GitHub Repository

  • Blog Preview Challenge

    This challenge was basically the same as the previous one, the difference is that in this one I should change the color of the title when you hover over it with the mouse, for that I used :hover. I also learned a bit more about display: grid;, and that's what I used to center the card.

    Useful Links:

    • Blog Preview - Frontend Mentor Solution Page
    • Blog Preview - GitHub Repository

  • Social Links Challenge

    In this challenge, I needed to insert a rounded photo, so I used border-radius: 50%; and it worked perfectly. I also needed to put my social networks inside containers that change the color when I hover the mouse over it.

    Useful Links:

    • Social Links - Frontend Mentor Solution Page
    • Social Links - GitHub Repository

  • Recipe Page Challenge

    This was a simple challenge, it consisted of creating a recipe page focusing on using semantic HTML tags like header, main and footer. For better mobile design, I also used @media

    Useful Links:

    • Recipe Page - Frontend Mentor Solution Page
    • Recipe Page - GitHub Repository

  • Product Preview Challenge

    This challenge was based on @media. The goal was to create two cards, one for the mobile version and the other for desktop, for this I used the mobile-first approach.

    Useful Links:

    • Product Preview - Frontend Mentor Solution Page
    • Product Preview - GitHub Repository

  • Four Cards Challenge

    Here, I needed to develop my knowledge further of display:grid; and principally of @media to create a responsive layout with the cards.

    Useful Links:

    • Four Cards - Frontend Mentor Solution Page
    • Four Cards - GitHub Repository
